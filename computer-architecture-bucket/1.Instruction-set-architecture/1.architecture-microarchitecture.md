# Architecture and Microarchitecture

## Architecture vs. Microarchitecture

### Architecture/Instruct Set Architecture:
- Programmer visible state (Memory & Register)
- Operations (Instructions and how they work)
- Execution Semantics (Interrupts)
- I/O
- Data Structures/Sizes

### Microarcitecture/Organization:
- Tradeoff on how to implement ISA for some metric (Speed, energy, cost)
- Examples: Pipeline depth, num of pipelines, cache size, silicon area, peak power, execution ordering, bus widths, ALU widths.

## IBM 360: A General- Purpose Register (GPR) Machine

### Processor State
- 16 General purpose 32-bit registers
- 4 floating point 64 bit registers
- A Program Status Word
