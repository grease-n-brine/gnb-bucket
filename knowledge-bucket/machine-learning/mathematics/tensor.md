# Tensor

## Definition:

Tensors are defined as multi-dimensional arrays that serve as the fundamental data structure for representing and manipulating data. A tensor is the fundamental data structure in ML used to represent all inputs, outputs, and parameters as N-dimensional arrays (scalars, vectors, matrices, and higher-order structures).

## Classes/Categories:

1. Scalars (0D),  
2. Vectors (1D),  
3. Matrices (2D),  
Batches of data(3D+)...

## Application:

### 1D Tensor (Vector):

A list of numbers.

    [2, 7, 9]

### 2D Tensor (Matrix):

A grid (rows × columns).

    [  
    [1, 2, 3],  
    [4, 5, 6]  
    ]

Shape: (2, 3), Access: x[i][j]  
x[1][2] = 6
